Aprendendo Listas

Existem ao todo 3 tipos de listas:

  1. Lista Ordenada
  2. Lista Desordenada
  3. Lista Mista
  4. Lista de Descrição

Listas Ordenadas

são listas que buscam preservar uma ordem numerica de se fazer alguma coisa. vamos considerar o cotidiano de uma pessoa:

  1. Acordar
  2. Ler a biblia
  3. tomar café
  4. tomar banho e escovar os dentes
  5. ir para a faculdade
  6. almoçar
  7. ir para casa
  8. jantar
  9. tomar banho
  10. dormir

Listas Desordenadas

são listas que não preservam uma ordem, consiste em tarefas que precisam ser feitas mas não necessariamente seguindo uma ordem, vamos considerar uma compra em um hortifruti:

  • Maracuja
  • Limão
  • Tomate
  • Abacaxi
  • Morango

Lista Descritiva

São Listas que apresentam um termo e sua respectiva descrição:

HTML:
Linguagem de marcação para Hipertexto, criada por Tin Berns Lee, na decada de 1990 para o desenvolvimento dos conteúdos pela internet.
CSS:
Linguagem para Estilização das páginas Web, visa chamar a atenção dos usuários para utilizar o conteúdo dos websites.
JavaScript:
Linguagem de programação, largamente utilizada pelos navegadores, o que facilita o desenvolvimento de websites cada vez mais iterativos e úteis para os usuários.
