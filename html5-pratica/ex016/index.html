Grouping Tags(tags Semânticas para agrupamento)

A tag 'header' é uma Grouping Tag(tag para agrupamento de outras tags) utilizada para exibir o cabeçalho de um website, menus, logo do website , em resumo, tudo o que representa o cabeçalho do website.

A tag 'nav', inserida dentro da Tag 'header', tambem é uma grouping tag que permite utilizar todo o sistema utilizada para a navegação dentro do próprio website

a tag 'main' posicionada abaixo da tag 'header' permite exibir o conteúdo principal do website.

A tag 'Footer' posicionada abaixo da tag 'main' permite exibir o rodapé do website
